1. What is DOM?what are the purpose of DOM?

DOM stands for Document Object Model. It is a programming interface for web documents that represents the web page as a tree-like structure of objects. The purpose of the DOM is to provide a way for web developers to access and manipulate the content and structure of web documents.

DOM is a crucial part of web development that provides a way to access and manipulate the content and structure of web documents, allowing developers to create dynamic and interactive web applications

2. How can you select DOM?

Select elements by tag name: var elements = document.getElementsByTagName("p"); Select elements by class name:var elements = document.getElementsByClassName("first-class"); Select elements by ID:var element = document.getElementById("his-name"); Select elements using CSS selectors: var elements = document.querySelectorAll("#first-element p");

3. What is event bubble?

Event bubbling is a mechanism in web development where events propagate through the DOM tree from the innermost element to the outermost element. It allows events to be handled at multiple levels of the DOM tree, but can also make event handling more complex and may require the use of stopPropagation() method in JavaScript to prevent an event from bubbling up to parent elements.

4. What is callback function? How it's work?

a callback function is a function that is passed as an argument to another function and is executed inside that function. The purpose of a callback function is to allow a function to call another function when a specific event occurs or a specific task is completed
